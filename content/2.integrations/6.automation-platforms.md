FilaForms provides a unified Automation API that powers Zapier, Make, and n8n integrations. All three platforms use the same REST Hook API endpoints with OAuth 2.0 authentication via Laravel Passport.

## API Endpoints

All endpoints use `auth:api` middleware (Laravel Passport):

| Method | Endpoint | Purpose |
|--------|----------|---------|
| `GET` | `/api/filaforms/automations/forms` | List user's forms |
| `POST` | `/api/filaforms/automations/subscribe` | Subscribe to form events |
| `DELETE` | `/api/filaforms/automations/unsubscribe/{id}` | Unsubscribe |
| `GET` | `/api/filaforms/automations/submissions` | Poll recent submissions |
| `GET` | `/api/filaforms/automations/auth/test` | Test authentication |

## Payload Format

```json
{
    "id": "01HX...",
    "form_id": "01HX...",
    "form_name": "Contact Form",
    "submitted_at": "2025-01-15T10:30:00+00:00",
    "field_name": "Jane Doe",
    "field_email": "jane@example.com"
}
```

## Zapier

### Zapier Developer Platform Configuration

- **Auth type:** OAuth v2
- **Authorization URL:** `https://{domain}/oauth/authorize`
- **Token URL:** `https://{domain}/oauth/token`
- **Scope:** `forms:read`
- **Trigger:** "New Form Submission" (REST Hook, instant)
- **Perform List:** `GET /api/filaforms/automations/forms`
- **Subscribe:** `POST /api/filaforms/automations/subscribe`
- **Unsubscribe:** `DELETE /api/filaforms/automations/unsubscribe/{id}`

### Laravel Passport Setup

Create an OAuth client for Zapier:

```bash
php artisan passport:client
```

Configure the client ID and redirect URI on the Zapier Developer Platform.

## Make

### Make Developer Hub Configuration

- **Auth type:** OAuth 2.0
- **Authorization URL:** `https://{domain}/oauth/authorize`
- **Access Token URL:** `https://{domain}/oauth/token`
- **Scope:** `forms:read`
- **Webhook module:** Subscribe/unsubscribe against the automation API endpoints

## n8n

### Environment Variable

```bash [.env]
FILAFORMS_N8N_INSTANCE_URL=https://your-n8n-instance.com
```

### n8n Custom Node Configuration

- **Credentials type:** OAuth 2.0
- **Authorization URL:** `https://{domain}/oauth/authorize`
- **Access Token URL:** `https://{domain}/oauth/token`
- **Scope:** `forms:read`
- **Trigger node:** Webhook subscription against the automation API endpoints
