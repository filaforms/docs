The Make integration connects FilaForms to Make's visual automation platform using the same unified Automation API as Zapier and n8n.

## Setup

### Make Developer Hub Configuration

- **Auth type:** OAuth 2.0
- **Authorization URL:** `https://{domain}/oauth/authorize`
- **Access Token URL:** `https://{domain}/oauth/token`
- **Scope:** `forms:read`
- **Webhook module:** Subscribe/unsubscribe against the automation API endpoints

### API Endpoints

Same endpoints as all automation platforms -- see the [Zapier docs](/integrations/zapier#api-endpoints) for the full reference.

| Method | Endpoint | Purpose |
|--------|----------|---------|
| `GET` | `/api/filaforms/automations/forms` | List forms |
| `POST` | `/api/filaforms/automations/subscribe` | Subscribe to events |
| `DELETE` | `/api/filaforms/automations/unsubscribe/{id}` | Unsubscribe |
| `GET` | `/api/filaforms/automations/submissions` | Poll submissions |
