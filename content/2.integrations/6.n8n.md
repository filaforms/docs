The n8n integration connects FilaForms to n8n using the unified Automation API. Since n8n is self-hosted, configure the instance URL in your environment.

## Configuration

```bash [.env]
FILAFORMS_N8N_INSTANCE_URL=https://your-n8n-instance.com
```

## Setup

### n8n Custom Node Configuration

- **Credentials type:** OAuth 2.0
- **Authorization URL:** `https://{domain}/oauth/authorize`
- **Access Token URL:** `https://{domain}/oauth/token`
- **Scope:** `forms:read`
- **Trigger node:** Webhook subscription against the automation API endpoints

### API Endpoints

Same endpoints as Zapier and Make -- see the [Zapier docs](/integrations/zapier#api-endpoints) for the full reference.
