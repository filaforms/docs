## Requirements

- **PHP:** 8.3+
- **Laravel:** 11+ or 12+
- **Filament:** 5.x

## Setup

::steps
### Install Package

```bash [Terminal]
composer require filaforms/core
```

### Publish Configuration

```bash [Terminal]
php artisan vendor:publish --tag="filaforms-config"
```

### Run Migrations

```bash [Terminal]
php artisan migrate
```

### Register the Plugin

Add `FilaFormsPlugin` to your Filament panel provider:

```php [app/Providers/Filament/AdminPanelProvider.php]
use FilaForms\Core\FilaFormsPlugin;

$panel
    ->plugins([
        FilaFormsPlugin::make(),
    ]);
```
