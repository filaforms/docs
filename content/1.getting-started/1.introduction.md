**FilaForms Integrations** is a Laravel package that connects your forms to external services -- payment processors, spreadsheets, automation platforms, webhooks, and more.

## Why FilaForms Integrations?

::card-group
  :::card
  ---
  icon: i-lucide-zap
  title: Minimal Code Per Integration
  ---
  Generic base classes handle HTTP requests, error handling, and field mapping. New integrations need only provider-specific code.
  :::

  :::card
  ---
  icon: i-lucide-map
  title: Field Selection & Mapping
  ---
  Checkbox-based field selection lets users choose which form fields to send. Includes metadata fields like submission date and IP address.
  :::

  :::card
  ---
  icon: i-lucide-key
  title: OAuth Token Management
  ---
  Encrypted token storage with automatic refresh. Supports Google Sheets and any OAuth 2.0 provider via the connection-based architecture.
  :::

  :::card
  ---
  icon: i-lucide-search
  title: Auto-Discovery
  ---
  Drop an integration class into the right directory and it's automatically registered. No manual configuration needed.
  :::
::

## Architecture

The package is built on a 3-interface system:

| Interface | Purpose | Examples |
|-----------|---------|----------|
| `IntegrationInterface` | Base contract for all integrations | All integrations |
| `ApiWriteIntegrationInterface` | API integrations with field mapping and OAuth | Google Sheets, CRMs |
| `ExternalIntegrationInterface` | Externally-configured integrations (info cards, no config form) | Zapier |

Generic base classes (`GenericApiIntegration`, `GenericOAuthIntegration`) eliminate code duplication across integrations.

### Two-Tier Connection Architecture

1. **Integration Connections** -- OAuth accounts created on the main Integrations page. One connection can be reused across multiple forms.
2. **Form Integrations** -- Per-form configuration that references a connection and stores integration-specific settings (spreadsheet selection, field mapping, etc.).

## Available Integrations

**Payments:** Stripe Connect (OAuth + platform payments)

**Storage:** Google Sheets (OAuth + Drive Picker)

**Automation:** Zapier (OAuth + REST Hook subscriptions via unified Automation API)

**Communication:** Webhooks (Standard Webhooks signing)
