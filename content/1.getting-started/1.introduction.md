**FilaForms** is a self-hosted form builder plugin for Filament. It provides a drag-and-drop form builder with live preview, conditional logic, multi-step forms, file uploads, email notifications, submission tracking, analytics, and integrations with external services.

## Core Features

  ---
  Visual form builder with live preview. 24+ field types including text, email, file upload, signature, line items, rich editor, and more.
  :::

  :::card
  ---
  icon: i-lucide-eye

  ---
  Show/hide fields based on other field values. Supports multiple conditions with AND/OR logic.
  :::

  :::card
  ---
  icon: i-lucide-mail

## Packages

| Package | Description |
|---------|-------------|
| `filaforms/core` | Form builder, submissions, notifications, analytics, public forms |
| `filaforms/integrations` | Stripe Connect, Google Sheets, Webhooks, Zapier, and automation API |

The integrations package is optional and requires the core package.

## Integrations Architecture

The integrations package is built on a 3-interface system:

| Interface | Purpose | Examples |
|-----------|---------|----------|
| `IntegrationInterface` | Base contract for all integrations | All integrations |
| `ApiWriteIntegrationInterface` | API integrations with field mapping and OAuth | Google Sheets, CRMs |
| `ExternalIntegrationInterface` | Externally-configured integrations (info cards, no config form) | Zapier |

Generic base classes (`GenericApiIntegration`, `GenericOAuthIntegration`) eliminate code duplication across integrations.

### Two-Tier Connection Architecture

1. **Integration Connections** -- OAuth accounts created on the main Integrations page. One connection can be reused across multiple forms.
2. **Form Integrations** -- Per-form configuration that references a connection and stores integration-specific settings (spreadsheet selection, field mapping, etc.).

## Available Integrations

**Payments:** Stripe Connect (OAuth + platform payments)

**Storage:** Google Sheets (OAuth + Drive Picker)

**Automation:** Zapier (OAuth + REST Hook subscriptions via unified Automation API)

**Communication:** Webhooks (Standard Webhooks signing)
