## Requirements

- **PHP:** 8.3+
- **Laravel:** 11+ or 12+
- **Filament:** 5.x

## Core Plugin

::steps
### Install Package

```bash [Terminal]
composer require filaforms/core
```

### Publish Configuration

```bash [Terminal]
php artisan vendor:publish --tag="filaforms-config"
```

### Run Migrations

```bash [Terminal]
php artisan migrate
```

### Register the Plugin

Add `FilaFormsPlugin` to your Filament panel provider:

```php [app/Providers/Filament/AdminPanelProvider.php]
use FilaForms\Core\FilaFormsPlugin;

$panel
    ->plugins([
        FilaFormsPlugin::make(),
    ]);
```
::

## Integrations Package (Optional)

::steps
### Install Package

```bash [Terminal]
composer require filaforms/integrations
```

### Publish Configuration

```bash [Terminal]
php artisan vendor:publish --tag="filaforms-integrations-config"
```

### Run Migrations

```bash [Terminal]
php artisan migrate
```

This creates the following tables:
- `integration_connections` -- OAuth accounts and credentials (encrypted)
- `form_integrations` -- Per-form integration configurations
- `integration_deliveries` -- Delivery log for tracking webhook/API calls

### Configure Integrations

Add credentials for the integrations you want to use in your `.env` file:

```bash [.env]
# Google Sheets
FILAFORMS_GOOGLE_SHEETS_ENABLED=true
GOOGLE_CLIENT_ID=your-client-id.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=your-client-secret
GOOGLE_API_KEY=your-api-key

# Stripe
STRIPE_PLATFORM_SECRET_KEY=sk_...
STRIPE_PLATFORM_PUBLISHABLE_KEY=pk_...
```
::

## Auto-Discovery

When `auto_discover` is `true` (default), integration classes are automatically found by scanning the `src/Integrations/` directory. No manual registration needed.

To disable auto-discovery and register integrations manually:

```php [config/filaforms-integrations.php]
'auto_discover' => false,
```
